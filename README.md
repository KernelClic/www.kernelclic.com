# KernelClic Corporate Website

This is the official corporate website for KernelClic SAS, developed with PHP, HTML5, CSS3, and JavaScript.

## Project Structure

- `index.php`: Home page
- `nosotros.php`: About Us
- `servicios.php`: Services
- `productos.php`: Products
- `contacto.php`: Contact
- `assets/`: Static assets (CSS, JS, Images)
- `includes/`: Reusable PHP components (header, footer)

## Setup

1. Clone the repository.
2. Run a local PHP server: `php -S localhost:8000`
3. Open `http://localhost:8000` in your browser.

## Technologies

- PHP 8+
- HTML5
- CSS3 (Flexbox/Grid, Custom Properties)
- JavaScript (ES6+)
